# 📒 정지우의 자바스크립트 코딩 치트시트

> 코드트리 문제 풀 때 막히면 여기 먼저! (검색 대신 이거 보기)

---

## 1️⃣ 입력 템플릿 (매번 복붙!)

```javascript
const fs = require("fs");
let input = fs.readFileSync(0).toString().trim().split(/\s+/);
// input[0], input[1]... 로 꺼내쓰기 (공백·줄바꿈 다 처리됨)
```

---

## 2️⃣ ⭐ 글자 vs 숫자 (제일 중요!)

```
"5"  → 따옴표 O = 글자 = 계산 X (그림 사과 🖼️)
 5   → 따옴표 X = 숫자 = 계산 O (진짜 사과 🍎)
```

- ★ **입력받은 건 무조건 "글자"** → 계산하려면 숫자로 바꿔야 함!
- ★ `"5" + 3 = "53"` (글자라 붙음!) / `5 + 3 = 8` (숫자라 더함)
- ★ 헷갈리면 그냥 `Number()` 씌우기 (숫자에 씌워도 안전)
- ★ 모르겠으면 `console.log(typeof 값)` → `string`(글자) / `number`(숫자)

---

## 3️⃣ 글자 → 숫자 변환

```javascript
Number("3.14")      // 3.14  (만능, 추천)
parseFloat("3.14")  // 3.14  (소수)
parseInt("3.14")    // 3     (정수만! 소수점 날아감 ⚠️)
```

---

## 4️⃣ 반올림 / 소수점 처리

```javascript
값.toFixed(2)      // 소수점 2자리 반올림 → "3.14" (결과는 글자!)
Math.round(2.5)    // 3   반올림
Math.floor(2.9)    // 2   내림 (무조건 ↓)  = Python의 //
Math.ceil(2.1)     // 3   올림 (무조건 ↑)
```

> 위치: **소수점 생기는 값**을 괄호 안에 → `Math.floor(total / 2)`
> 주의: `Math.floor(a)/2` ≠ `Math.floor(a/2)` (괄호 위치!)

---

## 5️⃣ 자르기 (split)

```javascript
값.split(' ')     // 공백으로
값.split('\n')    // 줄바꿈으로
값.split(/\s+/)   // 공백+줄바꿈+탭 전부 (만능)
값.split('')      // 글자 하나하나 (주의!)
```

---

## 6️⃣ 출력

```javascript
console.log(a);            // 한 줄
console.log(a, b);         // 여러 개 (띄어서)
console.log("점수 " + a);   // 글자랑 붙이기
```

---

## 7️⃣ 연산자 줄임말

```javascript
x += 5   // x = x + 5  (더해서 다시 넣기)
x -= 5   // x = x - 5
x *= 5   // x = x * 5
x /= 5   // x = x / 5
// 헷갈리면 긴 버전으로 풀어 써도 OK!
```

---

## 8️⃣ 자주 막히는 함정 🚨

- 입력 변환 안 함 → `"5"+2="52"` 사고
- `Number(" ") = 0`, `Number("") = 0` (빈 값은 0됨)
- `parseInt`로 소수 처리 → 소수점 날아감
- 변수명은 만든 이름과 똑같이! (`input` ≠ `Input`)
- `Math.floor(a)/2` ≠ `Math.floor(a/2)` (괄호 위치!)

---

## 9️⃣ 문제 푸는 5단계 루틴

1. **쪼개기** — 문제를 동작 단위로 (입력 / 계산 / 출력)
2. **도구 매칭** — 각 동작에 아는 도구 연결
3. **빠진 고리** — 안 맞는 부분 = 새로 배울 것
4. **한글 흐름** — 코드 전에 순서를 말로 먼저
5. **한 줄씩 + `console.log`로 확인**

---

## 🔟 신호 단어 → 도구

| 문제에 이런 말 | 쓸 도구 |
|---|---|
| "입력받아" | `Number(input[...])` |
| "반올림" | `toFixed` / `Math.round` |
| "내림 / 버림" | `Math.floor` |
| "올림" | `Math.ceil` |
| "공백으로 구분" | `split(/\s+/)` |
